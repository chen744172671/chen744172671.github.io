JAVA:java.lang.NullPointerException (int不能与Integer比较)
错误写法：
Integer s1 = null;
int s2 = 1;
boolean equals1 =  s2 == s1;
正确写法：
Integer s1 = null;
Integer s2 = 1;
boolean equals1 =  s2.equals(s1) ;
